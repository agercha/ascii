ИНСТРУКЦИИ:

dark mode is preferred for running this program

necessary packages:
    Pillow

terminal input:
    python krissas.py {image name} {resolution} {contrast factor}
    (suggested) python krissas.py kuhnya.jpg 200 1.25

image name 
    should be the name of an image file located in the "KRISSAS" folder
resolution 
    is the width of the outputted image, in pixels
contrast factor 
    keeps the image as is when it is 1, and increases or decreases contrast, respectively, as it is adjusted
    a negative contrast factor inverts the image
